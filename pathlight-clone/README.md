# PathLight Clone

PathLight (https://pathlight.co.jp/pathlight) のランディングページクローンです。

## 概要

エンタープライズ営業向けパワーチャート自動生成ツール「PathLight」のWebサイトを参考に実装したスタティックLPです。

## 技術構成

- HTML5 / CSS3（フレームワーク不使用）
- Google Fonts: Noto Sans JP, Outfit, Poppins
- レスポンシブ対応（768px以下）

## カラーパレット

| 用途 | カラーコード |
|---|---|
| 背景（メイン） | `#040d25` |
| 背景（セクション） | `#07112e` |
| アクセント | `#3a6cf4` |
| グラデーション | `#09173e → #1943b3` |

## 構成セクション

1. ナビゲーション（固定ヘッダー）
2. ヒーロー（パワーチャートデモビジュアル付き）
3. 統計（数値実績）
4. 課題提起
5. 機能紹介（6機能）
6. 活用シーン（3ステップ）
7. CTA
8. フッター

## 起動方法

```bash
# ローカルサーバーで開く（任意）
npx serve .
# または直接ブラウザで index.html を開く
open index.html
```
