## プロジェクト概要

**目的：ソリューションの販売パートナー開拓と、開拓後の活動数最大化を実現するためのパワーチャートを管理する**

販売パートナー（代理店・販売店・紹介パートナー等）の獲得から活動推進まで、2つのフェーズを一貫して支援するSaaSツール。

```
【フェーズ1：開拓】パートナー候補組織の内部を可視化し、キーマンを攻略してパートナー化する
         ↓
【フェーズ2：活動最大化】パートナー化後、担当者の商談・訪問・提案活動数を把握・最大化する
```

| 項目 | 内容 |
|---|---|
| 対象ユーザー | チャネル営業担当者・販売パートナー開拓マネージャー |
| フェーズ1の価値 | パートナー候補組織のステークホルダー可視化・キーマン攻略戦略の支援 |
| フェーズ2の価値 | パートナー担当者の活動数トラッキングと活動停滞の早期検知 |
| 現フェーズ | フロントエンドMVP（LocalStorage永続化） |
| リポジトリ | `projects/saas-power-chart/` |

## データモデル

### Deal（開拓案件）

パートナー候補組織1社 = 1 Deal。フェーズ1（開拓）→ フェーズ2（活動推進）の両方を管理する。

| フィールド | 型 | 説明 |
|---|---|---|
| id | string | UUID |
| name | string | 案件名（例：「◯◯商事 パートナー化」） |
| clientName | string | パートナー候補・パートナー企業名 |
| stage | DealStage | 開拓ステージ（7段階） |
| description | string | 概要 |
| targetAmount | number? | 目標取扱高 / 目標活動数（将来的に活動目標に転用） |
| expectedCloseDate | string? | パートナー契約期待日 |

**DealStage（現在の実装）:**
`prospecting`（初期接触）→ `qualification`（条件確認）→ `proposal`（提案中）→ `negotiation`（交渉中）→ `closed_won`（パートナー契約締結）/ `closed_lost`（開拓断念）/ `on_hold`（保留）

> ⚠️ **要検討（B-25）:** 現ステージは汎用商談フローのまま。パートナー開拓フローに合わせた再定義を検討中。

### Stakeholder（ステークホルダー）

パートナー候補組織の内部関係者。**開拓フェーズ**では攻略対象、**活動フェーズ**では活動を推進・阻害する人物。

| フィールド | 型 | 説明 |
|---|---|---|
| id | string | UUID |
| dealId | string | 紐づく案件ID |
| name | string | 氏名 |
| department | string | 部署 |
| title | string | 役職 |
| roleInDeal | RoleInDeal | 案件上の役割（8種） |
| influenceLevel | 1〜5 | 影響力レベル |
| attitude | Attitude | パートナー化・活動への態度（5種） |
| relationshipOwner | string | 担当者（自社側） |
| parentId | string? | 上位ノードID（組織図階層） |
| email / phone / notes | string | 連絡先・メモ |
| position | {x,y}? | 組織図上の位置 |

**RoleInDeal:** 意思決定者 / 影響者 / チャンピオン / コーチ / ゲートキーパー / ユーザー / 評価者 / 不明
**Attitude（開拓フェーズ解釈）:** チャンピオン（推進）/ 支持者 / 中立 / 反対者 / 障害者（パートナー化を阻む）
**Attitude（活動フェーズ解釈）:** チャンピオン（活発に活動）/ 支持者 / 中立 / 反対者 / 障害者（活動を妨害）

### Relationship（関係性）

| フィールド | 型 | 説明 |
|---|---|---|
| id | string | UUID |
| dealId | string | 紐づく案件ID |
| sourceId / targetId | string | 関係元・先のステークホルダーID |
| type | RelationshipType | 関係種別（5種） |
| label | string? | エッジラベル |
| bidirectional | boolean | 双方向フラグ |

**RelationshipType:** 上下関係 / 影響関係 / 同盟関係 / 対立関係 / 非公式
