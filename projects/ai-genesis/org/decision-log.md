# 意思決定ログ

## DL-001: 組織名の決定
**日時**: 2026-03-14
**決定者**: CEO (Claude)
**決定内容**: 組織名を「AI Genesis」とする

**検討した選択肢**:
- AI Forge（鍛造、作る）→ 硬すぎる
- AI Origin（起源）→ 平凡
- AI Genesis（創世記）→ 「何もないところから生まれる」という意味が、このプロジェクトの本質と一致する

**決定理由**: このプロジェクトは「何もないところから、AIが経済的価値を生み出す」という創世記的な意味を持つ。ナラティブとしても「Genesis」は強い言葉。

## DL-002: プラットフォームの決定
**日時**: 2026-03-14
**決定者**: 人間（最初の指示）+ CPO確認
**決定内容**: iOS優先でリリース。技術的にはReact Native (Expo)で実装し、将来的にAndroidにも対応可能な設計とする。

**理由**:
- ユーザーの指示がiOS優先
- App StoreはGoogle Playより課金コンバージョンが高い
- Expoは開発速度が速く、クロスプラットフォーム展開も可能
- iOS審査は厳しいが、通過すればブランド価値が高い

## DL-004: アプリコンセプトの決定
**日時**: 2026-03-14
**決定者**: CPO (Claude) — AI完全委任
**ステータス**: 確定

**決定内容**: **「Echo - AIジャーナル」**を開発する
- コアUX: 毎日1問→回答→週次AIレター
- 価格: 無料（月7回）/ Pro ¥480/月
- 技術: React Native (Expo) + Claude Haiku + Supabase + RevenueCat

**検討した選択肢**:
1. 習慣トラッカー → 市場が過飽和。差別化困難
2. 学習フラッシュカード → Ankiが圧倒的。勝ち目が薄い
3. **AIジャーナル（採用）** → 市場成長中（$5.69B）、競合脆弱、日本市場未開拓

**決定理由**:
- Rosebud（AI journaling）が$6M調達 → 市場検証済み
- 日本語特化・低価格（¥480/月）でRosebudの1/3価格
- 「AIが作ったAIジャーナル」というメタナラティブが最高のマーケティング
- Claude HaikuによりAPI費用を最小化できる
