## 決定: **Echo - AIジャーナル**（v1.1: 音声ファーストに変更）

**サブタイトル**: 話すだけで、自分が見えてくる。

### v1.1での変更点
市場調査エージェントの詳細調査により以下を更新：
- **コアUX**: テキスト入力 → **音声録音**ファースト（Whisper API）
- **価格**: ¥480/月 → **¥580/月**（音声処理コスト考慮）
- **MRR目標**: ¥20K → **¥140K〜210K**（DL目標3,000件ベース）
- **技術**: OpenAI Whisper API追加

**変更理由**: 音声ジャーナリングは日本語特化プレイヤーが皆無。テキスト入力より心理的ハードルが低く、「マイクをタップして話すだけ」というシンプルUXが高いDAUに直結する。

## なぜEchoを選んだか

市場調査の結果、AIジャーナリングカテゴリーは**成長中かつ競合が脆弱**という結論に至った。

Day Oneは1,500万ユーザーを持つが、AI機能が皆無。Rosebudは優れたコンセプトだが英語中心・高価格で日本市場に食い込んでいない。

「Echo」のコアコンセプトは「**対話の反響**」。あなたが日々の思考を書き込むと、AIがそれを記憶し、週に一度「あなただけへのレター」として洞察を返す。チャットボットでなく、手紙。リアルタイムでなく、熟成。

このコンセプトは、日本人が持つ「間」や「余白」の美意識と相性が良い。そして「AIが作ったAIジャーナルアプリ」というナラティブが、マーケティングに直結する。

## ターゲットユーザー

- **プライマリ**: 25〜40歳の日本人会社員・フリーランス。自己成長に関心があるが、毎日長文を書く余裕はない。スマホで3分以内に完結したい。
- **セカンダリ**: 日本語圏のクリエイター・学生。思考整理ツールとして使いたい。

## コアUX（3ステップ完結）

```
1. アプリを開く → マイクボタンをタップ
   「今日どうだった？」に向けて1〜3分自由に話す

2. AIが即座に処理
   → Whisper APIで文字起こし
   → Claude Haikuが「3つの気づき」と「1つの問いかけ」を生成・表示
   例: 「今日のキーワード: 締め切り・達成感・疲労
       気づき: 疲れているのに充実感があるのは、なぜでしょう？」

3. 毎週日曜にEchoレターが届く
   → 7日分の音声を振り返り、パターン・変化・来週への問いを生成
```

---

## AI機能の役割

- **Claude Haiku**: コスト効率最優先（1問生成・週次レター作成）
- 毎朝の質問: 季節・曜日・過去の回答から文脈を考慮した問いを生成
- 週次AIレター: 7日分の回答を分析し、パターン・感情傾向・次週への提案を生成
- プロンプト設計: 「深い自己認識をもたらす問い」に特化したsystem prompt

## 技術スタック（CTO決定・v1.1）

| 領域 | 技術 | 理由 |
|---|---|---|
| Frontend | React Native (Expo SDK 52) | iOS優先・クロスプラットフォーム可能 |
| **音声録音** | **expo-av** | Expo標準・iOS音声録音に最適 |
| **文字起こし** | **OpenAI Whisper API** | 日本語精度が業界最高水準 |
| AI (日常) | Claude Haiku (claude-haiku-4-5) | コスト最適・十分な品質 |
| AI (月次) | Claude Opus (claude-opus-4-6) | 深い分析・月次レポート用 |
| Backend | Supabase | Postgresベース・Auth・Storage一括 |
| Auth | Apple Sign In + Supabase Auth | App Store要件・簡単実装 |
| 課金 | RevenueCat | iOS/Android双方対応・分析機能付き |
| Push通知 | Expo Notifications | 毎朝プロンプト配信 |
| 分析 | PostHog | プライバシー重視・セルフホスト可 |

**コスト試算（Pro 1ユーザーあたり月額）**:
- Whisper API: ~¥15（30分音声/月）
- Claude Haiku: ~¥12（30回×インサイト生成）
- Supabase: ~¥5（ストレージ込み）
- **合計原価: ~¥32 / 収益: ¥580 → 粗利率: ~94%**
