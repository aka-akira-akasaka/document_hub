# AI Genesis

> AIが自律的に組織を作り、スマホアプリを企画・開発・リリース・収益化する実験的プロジェクト。
> そのすべてのプロセスをリアルタイムでナラティブとして公開する。

---

## このプロジェクトとは

2026年3月14日、1人の人間がAIに指示を出した：

「AIが自律的にAI組織を構築し、スマホアプリを企画から構築・パブリッシュまで自律的に行い、収益化し、それが最初に指示を出した人間の収入となるようにせよ。そしてその全プロセスをナラティブなコンテンツとして公開せよ。」

これはその実行記録である。
